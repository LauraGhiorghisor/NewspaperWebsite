How to make a sticky bar

  • Use JavaScript

  • Test the scrolling point, using an if condition that checks if the page has been scrolled past a certain number of pixels.

  • Example: if (window.scrollY > 200).

  • If the condition is met, then change the nav bar's position to fixed, with coordinates (0,0).

  • Reversely, when the user scrolls up, the navbar should be positioned static.

How to add a responsive set of images

  • Select a base image - the smallest size.

  • Add srcset attribute to the image.

  • The value of the srcset attribute is a list of image sources, separated by commas. Add image size to the source.

  • Tell the broswer how big to display the images by adding the "sizes" attribute.
